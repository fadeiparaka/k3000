# К-30 Event Bot

Бот для регистрации на афтерпати К-30 (27.02.2026)

## Описание

Telegram-бот для сбора регистраций на мероприятие с автоматическими напоминаниями и админ-функциями для рассылок.

## Требования

- Python 3.11+
- Telegram Bot Token
- Google Sheets API credentials (файл `credentials.json`)
- Google Sheet ID

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта.

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните `.env` файл:
   - `BOT_TOKEN` - токен вашего Telegram бота (получите у @BotFather)
   - `GOOGLE_SHEET_ID` - ID вашей Google таблицы (из URL)
   - `ADMINS` - список user_id администраторов через запятую
   - `EVENT_ANNOUNCE_LINK` - ссылка на анонс мероприятия в канале

5. Настройте Google Sheets:
   - Создайте сервисный аккаунт в Google Cloud Console
   - Скачайте `credentials.json` и положите в корень проекта
   - Создайте Google таблицу и поделитесь ею с email сервисного аккаунта (дайте права редактора)
   - Скопируйте ID таблицы из URL в `GOOGLE_SHEET_ID`

## Функции

### Для пользователей

- `/start` - начало работы с ботом, регистрация на афтерпати
- Процесс регистрации с валидацией имени и фамилии
- Автоматические напоминания:
  - 26 февраля в 22:00 - всем пользователям бота
  - 27 февраля в 10:00 - только зарегистрированным

### Для администраторов

- `/stats` - статистика регистраций и ссылка на таблицу
- `/post` - рассылка поста всем пользователям бота (ответьте `/post` на сообщение с текстом/медиа)
- `/addbutt`, `/delbutt`, `/editbutt` - управление динамическими inline-кнопками под сообщениями
- `/addmenu`, `/delmenu`, `/editmenu` - управление динамическими кнопками нижнего reply-меню

Динамические кнопки хранятся в SQLite. Inline-кнопки показываются под приветствием выше системной
кнопки `Я забыл свои вещи в К-30`. Системная кнопка потеряшек не редактируется через админку.
Ссылки поддерживаются только для inline-кнопок, потому что нижние reply-кнопки Telegram не открывают URL.

## Важные замечания

1. **Дедлайн регистрации**: 26 февраля 2026 в 23:30 по Москве
2. **База пользователей**: SQLite файл `users.db` создается автоматически
3. **Google Sheets**: Таблица должна иметь столбцы: User ID, Дата, Имя Фамилия, TG Username, Статус, Примечания
4. **Временная зона**: Все операции выполняются в таймзоне `Europe/Moscow`

## Безопасность

- Файл `credentials.json` не должен попадать в репозиторий (уже в `.gitignore`)
- Файл `.env` также не должен попадать в репозиторий
- Список админов хранится в `.env`

## Развертывание

Бот можно развернуть на:
- VPS (Debian + systemd)
- Railway
- Render
- Heroku
- Других PaaS платформах

Для production рекомендуется использовать systemd или supervisor для автозапуска и перезапуска бота.
